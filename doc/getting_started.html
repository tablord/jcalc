1 Tablord is a simple structured text authoring tool
Tablord structures the document in ELEMENTs. An ELEMENT is a block of text, a section, executable CODE or a structure that contains other elements. This is the Lego brick of Tablord. You will built you document out of this ELEMENTS.
1.1 Showing the structure and selecting an ELEMENT
When you move the mouse over the ELEMENT, you will see a small dotted line around the element that is hovered by the mouse. If you hover an ELEMENT that is contained inside another ELEMENT, all parent elements will also be dotted line in order to see the structure of the text

1.6 Add Class to your ELEMENTS
Tablord is negatively an HTML document. ELEMENTs are just normal tags (usually 
 but can be any tag) that have an ELEMENT class.
You can add other class in order to change the rendering of the ELEMENT.
In the future more class and user class will be introduced but to introduce that concept, click here to select this ELEMENT, and open the "properties" section.
just click on INFO or OK or WARNING or DANGER or -- in order to see the effect. you can also click on ADDRESS or NOTE
