1 create simple variable
the most simple for of variable declaration is just to take an ELEMENT an give it an "itemprop" name.
Just to get an idea, the next ELEMENT has an "itemprop" name. First just let the mouse hover that ELEMENT and you will see its itemprop name revealed
1123
in order to give an element an itemprop, select the element and open the properties section. then type a name in the itemprop input
exercise 1: add a new ELEMENT below. it will be written empty. then in the properties section of the menu, enter the itemprop name "demo2" the empty will now change to demo2. now select number: the element will now align to the right. finally type 456 in the element.
Doing so has created a variable "demo2" with a numerical value of 456
please do the exercise 1
2 use simple variable in function
having variable without using it would be useless. if you need variables it's because you want to calculate something with it. therefore you can declare functions. In fact functions are very similar to a variable. just instead of a text or a number, they have code that calculate a value. like in the example below. Just click on the element bellow to understand
1200
You can see that it has 1 line of code that say: demo1+77. as demo1 exists and its value is 123, it adds 77 which gives 200
At that stage, it is important to note that function can be declared in any order as the example bellow shows

4 declare a variable or a function inside a text
variables can also be declared inside the text using the 
{{variableName:value}} notation within the text. when run, this notation will first be transformed into a ELEMENT with itemprop = variableName and the content will be value. like here: 777  <-- if you go there with the mouse, you will see the dotted line showing this is an ELEMENT with demo7 as itemprop
similarly, 
{{functionName=code}} will be transformed into a function with itemprop=functionName and the code as the function code 888  
the SHOW class, will show the variableName in front of the value or the result of the function.
This can be helpful if you want to show the variable name inside the document.
exercise 4: go on the demo7: 777 variable declaration and select it. check / uncheck SHOW to see the effect
5 Be carefull with Javascript implicit data conversion 
Javascript is very flexible... but sometime a little bit too much. so the expression demo7+111 ends with  very different result depending on the type of demo7: if demo7 is a string "777" then javascript will assume that you want to do a string + string operation and then will convert 111 into a string and the result will be "777111"
if demo7 is a number Javascript will assume that you want to do a number + number operation and the result will be a number 888
